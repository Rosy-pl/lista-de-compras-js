# 🛒 Lista de Compras

Projeto desenvolvido com HTML, CSS e JavaScript para praticar manipulação do DOM, eventos, arrays e validação de input.

---

## 🚀 Funcionalidades
- Adicionar itens à lista
- Editar itens
- Remover itens
- Limpar toda a lista
- Baixar a lista em arquivo `.txt`
- Exibir mensagens de erro e sucesso

---

## 💻 Tecnologias utilizadas
- HTML
- CSS
- JavaScript

---

## 🎯 Objetivo
Este projeto foi desenvolvido com o objetivo de praticar:

- Estruturação de páginas com HTML
- Estilização e responsividade com CSS
- Manipulação do DOM com JavaScript
- Eventos de clique
- Uso de arrays para armazenar dados
- Criação dinâmica de elementos na tela

---

## 📌 Aprendizados
Durante o desenvolvimento deste projeto, pratiquei conceitos importantes do front-end, como:

- Captura de dados do input
- Atualização dinâmica de listas
- Edição e remoção de elementos
- Feedback visual para o usuário
- Geração de arquivo para download
